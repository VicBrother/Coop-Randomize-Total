# Coop-Randomize-Total
Mod para Super Mario 64 Coop Deluxe que aleatoriza objetos, niveles, música y más.


Coop Randomize Total

A feature-rich randomizer mod for Super Mario 64 Coop Deluxe, inspired by the original randomizer hack while expanding it with additional multiplayer-friendly mechanics, synchronization improvements, and quality-of-life features.

Features

Randomized Gameplay

- Randomized object locations.
- Randomized level entrances.
- Randomized background music.
- Randomized gameplay elements designed to make every playthrough unique.

Multiplayer Synchronization

- Deterministic seed generation and synchronization between players.
- Late-joining players automatically receive and apply the current randomization seed.
- Improved multiplayer stability and synchronization.
- Multiplayer-compatible star cutscene handling.

Gameplay Improvements

- Automatic handling of redirected level exits.
- Castle repositioning fixes after deaths or redirected exits.
- Improved object management and cleanup.
- Enhanced pause menu support.
- Additional quality-of-life improvements for Coop Deluxe.

Random Number Generation

- Custom deterministic RNG implementation.
- Stable seed behavior across multiplayer sessions.
- Consistent randomization for all connected players.

Difficulty

A Difficulty section exists in the mod, but it is currently not functional.

The system is planned for future development and may be implemented in a later update with different difficulty presets that affect the randomization experience.

Compatibility

- Super Mario 64 Coop Deluxe
- Multiplayer supported
- Designed to keep all players synchronized during gameplay

Credits

This project is based on the original SM64 Randomizer concept and expands it with additional multiplayer functionality, synchronization improvements, fixes, and new gameplay features for Coop Deluxe.
